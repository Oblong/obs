WIP buildbot - buildhost5

This buildbot is still being brought up. You probably want buildhost4 for now.

The main page is http://buildhost5.oblong.com:8010.

Try build results are visible there, but most users will just look at the links on the merge requests.

Note: if logs are not very legible, you may have run into a chrome font problem on ubuntu; try uninstalling ttf-mscorefonts-installer.

Apt server

Build results are available via an apt server. To add this apt server to your Ubuntu 16.04 or 18.04 system, install the package oblong-repo, then do 'sudo apt-get update'. You can then use 'sudo apt-get install ...' to install packages from this server.

However, this server is not yet complete. Don't do that unless you want to see just how incomplete it is.

Obs server

To install packages via obs on mac, see the obs README.

To temporarily switch to this server, do 'export MASTER=buildhost5.oblong.com' in the same shell window before running obs.
